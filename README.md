# 🏥 Healthcare Data Analysis Dashboard

## 📌 Project Overview
The **Healthcare Data Analysis Dashboard** project focuses on analyzing hospital and patient records to identify trends related to **patient admissions, billing amount, medical conditions, and insurance providers**.

This project helps healthcare organizations gain meaningful insights from raw patient data through **data analysis and interactive dashboard visualization**, enabling faster and better decision-making.

---

## 🎯 Objective

✔️ Analyze healthcare patient records  
✔️ Track hospital billing information  
✔️ Identify common medical conditions  
✔️ Monitor insurance provider usage  
✔️ Build an interactive Power BI dashboard for decision-making

---
